# dls-AI - 多邻省 AI 学习助手

## 📚 项目简介

**dls-AI** 是一个多邻省风格的 AI 语言学习助手项目，专注于方言学习与翻译。本项目采用"过程导向"的开发模式，不仅包含核心应用实现，还完整记录了开发过程中的文档、评测、日志和反思。

### 核心功能

- **AI 对话聊天**: 支持配置不同的 AI 模型（OpenAI API 兼容接口或本地 Ollama）
- **方言翻译**: 
  - 上海话 ⇄ 中文互译
  - 粤语 ⇄ 中文互译
  - 上海话 ⇄ 粤语互译
- **双模式运行**:
  - AI 模型模式：调用大语言模型进行高质量翻译
  - 本地词典模式：无 API 时使用内置基础词典兜底

## 🚀 快速开始

### 方式一：直接打开（推荐）

```bash
cd dls-ai-main
open index.html  # macOS
# 或直接双击打开 index.html
```

### 方式二：使用本地服务器

```bash
cd dls-ai-main
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`

## ⚙️ 配置说明

在应用界面中可配置以下参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| API Key | 空 | AI 服务密钥（可选） |
| Base URL | `http://localhost:11434` | AI 服务地址 |
| Model | `qwen2.5:7b` | 使用的模型名称 |

### 常用配置示例

**使用 OpenAI:**
- Base URL: `https://api.openai.com/v1`
- Model: `gpt-4o-mini`

**使用本地 Ollama:**
- Base URL: `http://localhost:11434`
- Model: `qwen2.5:7b`

配置会自动保存到浏览器的 `localStorage`。

## 🎯 使用场景

### 1. 方言学习
- 输入中文，获取上海话/粤语翻译
- 输入方言，理解其含义
- 与 AI 助手进行方言学习对话

### 2. 语言对比研究
- 对比上海话和粤语的表达差异
- 探索不同方言间的转换规律

### 3. 无网络环境
- 不配置 API Key 时，自动使用本地词典兜底
- 适合基础词汇的快速查询

## 📝 开发说明

### 核心组件

- [`app.js`](dls-ai-main/app.js): 包含对话、翻译、配置管理等核心逻辑
- [`index.html`](dls-ai-main/index.html): 应用主界面
- [`styles.css`](dls-ai-main/styles.css): 样式定义

### 支持的翻译方向

```javascript
zh_to_sh    // 中文 → 上海话
sh_to_zh    // 上海话 → 中文
zh_to_yue   // 中文 → 粤语
yue_to_zh   // 粤语 → 中文
sh_to_yue   // 上海话 → 粤语
yue_to_sh   // 粤语 → 上海话
```
