# reflection 使用说明

本目录用于存放每次更新与里程碑的复盘反思。

包含：

- `lesson_reflection_01.md` ... `lesson_reflection_15.md`
- `milestone_reflection_m1.md` ... `milestone_reflection_m4.md`
- `final_reflection.md`

每份反思建议覆盖：

1. 本次目标是否达成
2. 关键结果与证据
3. 失败案例与根因
4. 下一步行动清单

最低提交标准：

1. 每个里程碑至少 1 份复盘
2. 最终阶段至少 1 份总反思
3. 反思必须包含可执行改进动作
